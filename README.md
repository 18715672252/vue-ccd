# simpleProjectDemo

## 分支使用说明
```
npm install //安装依赖包
npm run serve //运行
npm run build //打包
```
## 代码规范
```
+ 小驼峰命名：文件夹、props使用时
+ 大驼峰命名：自定义的组件引入、使用，组件的name属性
+ 短横线：文件名、props传入时、class、id、框架级别组件
```
